LLM Inference Performance

Overview

My current work at Zyphra focuses on inference performance and ML infrastructure for large models, especially AMD GPU training and serving environments. The work combines training control-plane systems, checkpoint recovery, GPU systems debugging, benchmark design, and production reliability.

I contributed to AMD MI355X inference benchmarking and full-stack optimization, characterizing TTFT, output throughput, kernel behavior, RCCL/HIP stack performance, communication patterns, and high-throughput serving metrics across large-model inference workloads.

Adjacent work includes Zookeeper, Zyphra's internal control plane for large-scale LLM training, plus distributed checkpoint recovery systems for pairwise NVMe replication, failed-node replacement, object-store checkpoint repair, and operational workflows for Slurm, Prometheus, NFS/NVMe, and GPU cluster storage pressure.

Technical Areas

  • AMD MI355X inference benchmarking
  • TTFT, output-throughput, and serving-metric analysis
  • Kernel behavior, RCCL/HIP stack performance, and communication-pattern analysis
  • GPU systems reliability and root-cause analysis
  • Distributed checkpoint validation and recovery
  • Slurm, Prometheus, NFS/NVMe, and object-store-backed infrastructure
